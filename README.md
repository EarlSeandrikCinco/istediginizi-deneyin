## ğŸš§ Features

This application includes various features to explore distributed systems:

- **Distributed Lock**: Prevent race conditions by implementing distributed locking mechanisms.
  
- **Distributed Transaction**: Manage multi-service operations using transaction strategies like Saga Pattern and Two-Phase Commit.
